AWS General Immersion Day - Advanced Workshop Summary

Before Starting the Workshop

Prior to diving into the workshop modules, participants are encouraged to familiarize themselves with the basic concepts of AWS, ensuring a smooth progression through the advanced topics.

Advanced Modules - Web Application

Network – Amazon VPC

Participants learn to create a Virtual Private Cloud (VPC) and have the option to create VPC endpoints for enhanced security and connectivity.

Compute – Amazon EC2

This module delves into Amazon Elastic Compute Cloud (EC2), covering advanced aspects of compute resources management within AWS.

Database – Amazon Aurora

The workshop explores Amazon Aurora, a powerful and scalable relational database service, providing insights into advanced database configurations.

Storage – Amazon S3

A comprehensive study of Amazon Simple Storage Service (S3) is conducted, including advanced storage configurations. The module concludes with resource cleanup guidelines.

Advanced Modules - Cost Monitoring and Observability

Getting Started

Participants receive an introduction to cost management, setting the stage for an in-depth exploration of cost monitoring and observability.

Introduction to Cost Management

This section provides insights into effectively managing costs within the AWS environment.

Content Preferences

The workshop caters to participants with a preference for advanced content, providing a deeper dive into AWS services and capabilities. By the end of the workshop, participants are expected to have gained advanced skills in web application development, network configuration, compute resources management, database administration, storage solutions, and cost monitoring, enabling them to harness the full potential of AWS for their business needs.
